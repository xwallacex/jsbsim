      56             : /** Lookup table class.
      57             : Models a one, two, or three dimensional lookup table for use in aerodynamics
      58             : and function definitions.
      59             : 
      60             : For a single "vector" lookup table, the format is as follows:
      61             : 
      62             : @code
      63             : <table name="property_name">
      64             :   <independentVar lookup="row"> property_name </independentVar>
      65             :   <tableData>
      66             :     key_1 value_1
      67             :     key_2 value_2
      68             :     ...  ...
      69             :     key_n value_n
      70             :   </tableData>
      71             : </table>
      72             : @endcode
      73             : 
      74             : The lookup="row" attribute in the independentVar element is option in this case;
      75             : it is assumed that the independentVar is a row variable.
      76             : 
      77             : A "real life" example is as shown here:
      78             : 
      79             : @code
      80             : <table>
      81             :   <independentVar lookup="row"> aero/alpha-rad </independentVar>
      82             :   <tableData>
      83             :    -1.57  1.500
      84             :    -0.26  0.033
      85             :     0.00  0.025
      86             :     0.26  0.033
      87             :     1.57  1.500
      88             :   </tableData>
      89             : </table>
      90             : @endcode
      91             : 
      92             : The first column in the data table represents the lookup index (or "key").  In
      93             : this case, the lookup index is aero/alpha-rad (angle of attack in radians).
      94             : If alpha is 0.26 radians, the value returned from the lookup table
      95             : would be 0.033.
      96             : 
      97             : The definition for a 2D table, is as follows:
      98             : 
      99             : @code
     100             : <table name="property_name">
     101             :   <independentVar lookup="row"> property_name </independentVar>
     102             :   <independentVar lookup="column"> property_name </independentVar>
     103             :   <tableData>
     104             :                 {col_1_key   col_2_key   ...  col_n_key }
     105             :     {row_1_key} {col_1_data  col_2_data  ...  col_n_data}

     107             :     { ...     } {...         ...         ...  ...       }
     108             :     {row_n_key} {...         ...         ...  ...       }

     110             : </table>
     111             : @endcode
