      58             : /** Models the EOM and integration/propagation of state.
      59             :     The Equations of Motion (EOM) for JSBSim are integrated to propagate the
      60             :     state of the vehicle given the forces and moments that act on it. The
      61             :     integration accounts for a rotating Earth.
      62             : 
      63             :     Integration of rotational and translation position and rate can be
      64             :     customized as needed or frozen by the selection of no integrator. The
      65             :     selection of which integrator to use is done through the setting of
      66             :     the associated property. There are four properties which can be set:
      67             : 
      68             :     @code
      69             :     simulation/integrator/rate/rotational
      70             :     simulation/integrator/rate/translational
      71             :     simulation/integrator/position/rotational
      72             :     simulation/integrator/position/translational
      73             :     @endcode
      74             : 
      75             :     Each of the integrators listed above can be set to one of the following values:
      76             : 
      77             :     @code
      78             :     0: No integrator (Freeze)
      79             :     1: Rectangular Euler
      80             :     2: Trapezoidal
      81             :     3: Adams Bashforth 2
      82             :     4: Adams Bashforth 3
      83             :     5: Adams Bashforth 4
      84             :     @endcode
      85             : 
      86             :     @author Jon S. Berndt, Mathias Froehlich, Bertrand Coconnier
      87             :   */
