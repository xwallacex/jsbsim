     106             : Definition of the gas cell configuration file parameters:
     107             : - <b>type</b> -
     108             :     One of HYDROGEN, HELIUM or AIR.
     109             : - <b>location</b> -
     110             :     Location of cell center in the aircraft's structural frame.
     111             :     Currently this is were the forces of the cell is applied.
     112             : - <b>{x|y|z}_radius</b> -
     113             :     Radius along in the respective direction (both ends).
     114             : - <b>{x|y|z}_width</b> -
     115             :     Width in the respective direction.
     116             :     <b>NOTE:</b> A 'x', 'y', 'z'-radius/width combination must be specified.
     117             : - <b>fullness</b> -
     118             :     Initial fullness of the cell, normally [0,1],
     119             :     values >1 initialize the cell at pressure.
     120             : - <b>max_overpressure</b> -
     121             :     Maximum cell overpressure (excess is automatically valved off).
     122             : - <b>valve_coefficient</b> -
     123             :     Capacity of the manual valve. The valve is
     124             :     considered to be located at the top of the cell.
     125             :     The valve coefficient determine the flow out
     126             :     of the cell according to:
     127             :     <i>dVolume/dt = ValveCoefficient * DeltaPressure</i>.
     128             : - <b>heat</b> -
     129             :     Zero or more FGFunction:s describing the heat flow from
     130             :     the atmosphere into the gas cell.
     131             :     Unit: [lb ft / (sec Rankine)].
     132             :     If there are no heat transfer functions at all the gas cell
     133             :     temperature will equal that of the surrounding atmosphere.
     134             :     A constant function returning 0 results in adiabatic behaviour.
     135             : - <b>ballonet</b> -
     136             :     Zero or more ballonets, i.e. air bags inside the gas cell.
     137             :     Ballonets are used to maintain the volume of the gas cell
     138             :     and keep its internal pressure higher than that of the
     139             :     surrounding environment.
     140             :   - <b>location</b> -
     141             :       Location of ballonet center in the aircraft's structural frame.
     142             :   - <b>{x|y|z}_radius</b> -
     143             :       Radius along in the respective direction (both ends).
     144             :   - <b>{x|y|z}_width</b> -
     145             :       Width in the respective direction.
     146             :   - <b>max_overpressure</b> -
     147             :       Maximum ballonet overpressure (excess is automatically valved off).
     148             :   - <b>valve_coefficient</b> -
     149             :       Capacity of the exit valve between the ballonet

     151             :       determine the flow out of the cell according to:
     152             :       <i>dVolume/dt = ValveCoefficient * DeltaPressure</i>.
     153             :   - <b>heat</b> -

     156             :       Unit: [lb ft / (sec Rankine)]
